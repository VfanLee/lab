inherit 关键字用于设置 CSS 属性为它的父元素的值，也就是继承父元素的样式。例如：color: inherit 就相当于把 color 属性设置为和父元素一样的颜色。

initial 关键字用于设置 CSS 属性为它的默认值，也就是浏览器的初始样式。例如：color: initial 就相当于 color: black，因为 black 是 color 属性的默认值。

unset 关键字用于设置 CSS 属性为不固定的值，也就是根据属性的继承特性来决定。
  - 如果该属性是默认继承属性，那么 unset 等同于 inherit；例如：color: unset 就相当于 color: inherit，因为 color 是可继承属性。
  - 如果该属性是非继承属性，那么 unset 等同于 initial。例如：border: unset 就相当于 border: initial，因为 border 是不可继承属性。

revert 关键字用于设置 CSS 属性为它的原始值，也就是恢复到之前的样式。
  如果该属性有用户代理（浏览器）的样式，那么 revert 等同于用户代理的样式。例如：color: revert 就相当于恢复到浏览器默认的颜色。
  如果该属性没有用户代理的样式，那么 revert 等同于 initial。例如：border: revert 就相当于恢复到没有边框的样式。
